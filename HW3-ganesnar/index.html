Web Dev Concepts

Definitions of terms:

Website: A website is a location on the internet that holds multiple files and uses them to display information.

Web Application: A web application provides functionality for the files on a website.

Web Browser: A web browser is an online platform that can be used to access websites.

Web Server: A web server is a system that handles transferring files made to be published from the author to users on the internet (via the web browser) and helps to communicate and exchange information online.

HTTP Request: An HTTP request is a message sent to the server by the user in order to request a certain document or resource.

HTTP Response: An HTTP response is a message sent to the user by the server to provide the user with the document or resource they've requested.

Frontend Design:

Front end design refers to what the user sees when they interact with a website or online application. This referes to the buttons, visuals, and other content.

Effective:
This refers to how useful the website is in helping the user fulfill the reason they visited the applciation.
Efficient:
This refers to completing the task with the least number of steps in the least amount of time with the most clarity.
Easy to Navigate:
This refers to how users can easily find the resources they need on the website, and also how easily they can remember it.
Error-free:
This refers to avoiding issues regarding accessibilty and availability. This can include preventing bugs that might be caused by a user's inputs, or even spelling issues on the website.
Enjoyable or Engaging:
This refers to how the website fits the user's unique taste in visuals and content.

We need page layout tags because they determine how the website will appear to the user.

Different Tags and What They Mean
Tag Description
header The header tag denotes the names used in an app/site. It typically has the name, publisher, and marketing slogan.
nav This tag can be used multiple times. It takes the user from the current page to a differnet page. It is typically used in a main menu, search, stories, etc.
main This tag is the primary block that holds content.
section This tag groups content together, usually by a common theme because the content cannot stand alone.
article This tag is usually used inside the section tag and refers to a single topic related to the theme of the section. It is usually denoted with a second-level headline, or whatever is below the one used for the section.
footer This tag usually holds the legal and/or contact information and links to important pages. It should also include the owner's copyright stataement. This includes the copy right symbol (©), year of publication, and the owner's name.
